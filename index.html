Privacy Policy

Last Updated: January 15, 2024

Overview

Screenshot Vault is committed to protecting your privacy. This privacy policy explains how we handle your data.

Data Collection

We do not collect any personal data.

Screenshot Vault:

  • Does NOT collect personal information
  • Does NOT track your browsing activity
  • Does NOT send data to external servers
  • Does NOT use analytics or telemetry
  • Does NOT share data with third parties

Data Storage

Local Storage Only

All screenshots and data are stored locally on your device using Chrome's built-in storage API (chrome.storage.local). This data:

  • Remains on your device
  • Is never uploaded to any server
  • Is only accessible by you
  • Can be deleted at any time

What We Store Locally

  • Screenshot images (as data URLs)
  • Screenshot metadata (timestamp, page title, page URL)
  • Your editing annotations

Data Retention

  • Data persists until you manually delete it
  • Uninstalling the extension removes all stored data
  • You can clear all screenshots from the dashboard at any time

Permissions Explained

Permission Purpose Data Access
activeTab Capture screenshots of the current tab Temporary access to capture visible tab content as image
storage Save screenshots locally on your device Local browser storage only
tabs Get page title and URL for screenshot metadata Current tab URL and title only
scripting Inject snipping tool overlay and show notifications No data access, only UI injection
clipboardWrite Copy screenshots to clipboard Write-only, no clipboard reading

Third-Party Services

Screenshot Vault does not use any third-party services, including:

  • No analytics (Google Analytics, etc.)
  • No crash reporting
  • No advertising networks
  • No cloud storage services
  • No external APIs

Network Activity

Screenshot Vault makes zero network requests. All functionality works completely offline.

Children's Privacy

Screenshot Vault does not knowingly collect any information from anyone, including children under 13.

Your Rights

You have complete control over your data:

  • Access: View all your screenshots in the dashboard
  • Delete: Remove individual screenshots or clear all data
  • Export: Download screenshots as PNG files
  • Control: All data stays on your device

Data Security

Your screenshots are protected by:

  • Chrome's built-in security features
  • Local storage encryption (provided by Chrome)
  • No network transmission
  • No external access
